# 🧠 1. System Architecture (Agent-Based Scraper)

Think in **modular agents**, not a single script.

```
User Input
   ↓
Search Agent (Google/Bing/DuckDuckGo)
   ↓
Link Filtering Agent (Find relevant pages)
   ↓
Crawler Agent (Visit pages, scroll, render JS)
   ↓
PDF Extraction Agent (Detect & download PDFs)
   ↓
Validation Agent (Check year + exam match)
   ↓
```

---

## Step 2: Search Agent

Use multiple sources to avoid blocking:

```python
from duckduckgo_search import DDGS

def search_links(query):
    results = []
    with DDGS() as ddgs:
        for r in ddgs.text(query, max_results=10):
            results.append(r['href'])
    return results
```

---

## Step 3: Filter Relevant Links

Filter by keywords:

```python
KEYWORDS = ["pdf", "question paper", "previous year"]

def is_relevant(url):
    return any(k in url.lower() for k in KEYWORDS)
```

---

## Step 4: Crawler Agent (Playwright)

Handles:

* Dynamic pages
* "Click here to download"
* Infinite scroll

```python
from playwright.sync_api import sync_playwright

def get_page_links(url):
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=True)
        page = browser.new_page()
        page.goto(url, timeout=60000)

        page.wait_for_timeout(3000)

        links = page.eval_on_selector_all(
            "a", "elements => elements.map(e => e.href)"
        )

        browser.close()
        return links
```

---

## Step 5: PDF Extraction

```python
def extract_pdf_links(links):
    return [link for link in links if link.endswith(".pdf")]
```

---

## Step 6: Validation Agent (Important)

Avoid wrong downloads:

```python
import re

def is_valid_pdf(url, exam_name, year):
    return (
        exam_name.lower() in url.lower()
        and str(year) in url
    )
```

---

## Step 7: Download + Store

```python
import os
import requests

def download_pdf(url, exam, year):
    folder = f"data/{exam}/{year}"
    os.makedirs(folder, exist_ok=True)

    file_name = url.split("/")[-1]
    path = os.path.join(folder, file_name)

    r = requests.get(url)
    with open(path, "wb") as f:
        f.write(r.content)
```

## 🔁 8. Main Orchestrator

```python
for year in range(start_year, end_year + 1):
    queries = generate_queries(exam, year)

    for query in queries:
        links = search_links(query)

        for link in links:
            if not is_relevant(link):
                continue

            page_links = get_page_links(link)
            pdfs = extract_pdf_links(page_links)

            for pdf in pdfs:
                if is_valid_pdf(pdf, exam, year):
                    download_pdf(pdf, exam, year)
```
